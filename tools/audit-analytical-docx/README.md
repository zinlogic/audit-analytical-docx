# audit-analytical-docx

## Cómo actualizar

Para obtener la última versión:

```bash
git pull --ff-only
cd tools/<nombre-del-skill>
source .venv/bin/activate
pip install -r requirements.txt
```

Si hay problemas con dependencias:

```bash
rm -rf .venv
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```
